Écriture et lecture de fichiers de données · Python & JSON

Description

Cette SAé avait pour objectif de transformer un fichier JSON brut contenant des mesures de concentrations de polluants dans l'air ambiant à Poitiers (SO2, NOx, NO2, PM10, PM2.5, O3, benzène, CO) en un fichier CSV propre et exploitable dans Excel.

Le script Python devait extraire uniquement les lignes complètes, reformater les dates en format français, utiliser le point-virgule comme séparateur et gérer correctement les accents.

Compétences acquises

Python Fichiers JSON Export CSV Nettoyage de données Bonnes pratiques de code

Outils & Technologies

  • Python — script de transformation de données
  • Format JSON — lecture et parsing
  • Format CSV — export et ouverture Excel

Ce que j'ai appris

J'ai mis en place un programme assez simple qui permettait, sur ce jeu de données, de le transformer en fichier exploitable dans Excel.
