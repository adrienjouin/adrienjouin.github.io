# Portfolio BUT Science des Données

## Structure du dossier

```
portfolio/
├── index.html              ← Page d'accueil
├── style.css               ← Styles partagés (ne pas modifier)
├── images/                 ← Mets toutes tes images ici
│
└── pages/
    ├── apropos.html        ← À propos, CV, bilan
    ├── saes.html           ← Liste de toutes les SAé
    ├── contact.html        ← Tes infos de contact
    │
    └── sae/                ← Une page par SAé
        ├── demographie.html
        ├── python-json.html
        ├── vba.html
        ├── bdd.html
        ├── reporting-dataviz.html
        ├── kpi.html
        ├── stats-regression.html
        ├── stats-inf.html
        └── dataviz.html
```

## Comment personnaliser

### Ton nom (index.html et apropos.html)
Cherche `Prénom Nom` dans le fichier et remplace-le.

### Tes infos de contact (pages/contact.html)
Remplis les champs LinkedIn, e-mail directement dans le navigateur (clique dessus).

### Ajouter du contenu dans chaque SAé
Ouvre la page dans le navigateur → clique sur les zones de texte → écris directement.

### Ajouter des images
1. Mets tes images dans le dossier `images/`
2. Sur chaque page SAé, clique sur la zone image pour en choisir une

### Héberger sur les serveurs SD
Dépose l'intégralité du dossier `portfolio/` sur le serveur.
L'URL d'accès sera : `http://serveur-sd/~tonlogin/portfolio/`
